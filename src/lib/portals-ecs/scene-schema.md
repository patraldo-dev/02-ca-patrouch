# Portal Scene Config Schema

Generated by Workers AI from published writings, consumed by bottle-world.js ECS.

## JSON Structure

```json
{
  "portal_id": "fiesta",
  "generated_at": "2026-06-23T15:00:00Z",
  "source_writings": ["uuid1", "uuid2", "..."],
  "atmosphere": {
    "mood": "festive",
    "intensity": 0.8,
    "ambient_color": "#ff6b35",
    "fog_density": 0.02,
    "light_intensity": 0.6
  },
  "palette": {
    "primary": "#ff6b35",
    "secondary": "#ffcc02",
    "accent": "#ff006e",
    "crystal_colors": ["#ff6b35", "#ffcc02", "#ff006e", "#8338ec"]
  },
  "decorations": {
    "particle_count": 40,
    "particle_style": "sparkle",
    "pillar_count": 6,
    "pillar_height": 2.5,
    "spiral_count": 3,
    "spiral_speed": 0.5,
    "halo_radius": 1.2,
    "halo_pulse_speed": 0.6
  },
  "crystals": [
    {
      "text": "Bajo el crepúsculo, el sonido de una campana susurra secretos",
      "position": [1.5, 1.2, -2.0],
      "color_index": 0,
      "scale": 1.0
    }
  ],
  "ambient_texts": [
    "¿Has visto cómo va vestido el Cap'n Crunch?",

  ],
  "spatial_layout": {
    "crystal_ring_radius": 2.0,

    "tab_count": 9
  }

```

## Field Semantics

### atmosphere
- **mood**: string label (festive, contemplative, melancholic, cosmic, etc.)
- **intensity**: 0.0-1.0 — drives particle density and animation speed
- **ambient_color**: hex color for ambient lighting tint
- **fog_density**: 0.0-0.1 — atmospheric fog
- **light_intensity**: 0.0-1.0 — point light brightness

### palette
Overrides portal defaults with AI-selected colors derived from writings.
- **crystal_colors**: array of hex colors, cycled across bottle markers

### decorations
Controls ECS entity counts and parameters.
- **particle_style**: "sparkle" | "dust" | "ember" | "bubble" | "snow"
- All counts are clamped to performance limits (max 60 particles, 8 pillars, 5 spirals)

### crystals
Text snippets extracted from writings, placed as floating crystal markers.
- **text**: excerpt from a published writing (max 200 chars)
- **position**: [x, y, z] in local space relative to user
- **color_index**: index into palette.crystal_colors
- **scale**: size multiplier

### ambient_texts
Short phrases (max 100 chars) that drift as particle-like text motes.

### spatial_layout
Macro geometry of the scene.
- **crystal_ring_radius**: distance from user to crystal ring
- **crystal_elevations**: y-positions for tiered placement
- **tab_orbit_radius**: distance for portal tab navigation
